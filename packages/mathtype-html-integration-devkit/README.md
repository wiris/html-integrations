# MathType Web Integration JavaScript SDK

## Install instructions

Install dependencies:

```bash
$ npm install
```

Compile using npm packages:

```bash
$ npm run build
```

Compile using local packages using 'lerna':

```bash
$ npm run build-dev
```

Lint:

```bash
$ npm run lint
```

Test:

```bash
$ npm run test
```

Generate documentation:

```bash
$ npm run build-docs
```
